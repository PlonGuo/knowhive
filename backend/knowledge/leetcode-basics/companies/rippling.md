# Rippling

## Interview Focus Areas

- Fullstack intern technical interview, emphasis on practical problem solving
- Heavy focus on **dynamic programming variants** (5 out of 6 known problems)
- **Graph algorithms** are a key area based on Glassdoor and LeetCode discussion reports
- Difficulty: Medium to Hard

## Graph Algorithm Focus (Verified from Interview Reports)

### Confirmed Interview Topics

1. **Standard Dijkstra implementation** - commonly the first problem
2. **Undirected graph longest path** - BFS/DFS application
3. **Sliding window + binary search combination** - graph + other algorithm combinations

### Must-Know Graph Algorithms

- BFS - unweighted shortest path
- DFS - graph traversal and connectivity
- Dijkstra - non-negative weight single-source shortest path (key focus)

### Related LeetCode Problems

- LC 743 - Network Delay Time (classic Dijkstra)
- LC 778 - Swim in Rising Water (Dijkstra variant)
- LC 310 - Minimum Height Trees (tree diameter / longest path)

## Key Patterns

- **DP variants**: interval DP, digit DP, rerooting DP - understand how to identify and apply each type
- **Graph algorithms**: Dijkstra is particularly important; practice both standard and variant implementations
- **Problem transformation**: converting real-world scenarios into algorithmic models

## Preparation Recommendations

1. Master DP classification - distinguish characteristics of different DP types
2. Practice Dijkstra thoroughly - standard implementation and common variants
3. Understand problem transformation - how to model real problems as DP or graph problems
4. Work through related problems - build pattern recognition across similar problem types
