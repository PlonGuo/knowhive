# LC 2385 - Amount of Time for Binary Tree to Be Infected

## Problem

Given the root of a binary tree and an integer `start`, at minute 0 a node is infected. Each minute, infected nodes infect all adjacent nodes. Return the number of minutes needed to infect the entire tree.

## Approach

### Core Idea

- Infection spreads like BFS in an undirected graph
- Convert the binary tree to an undirected graph (need parent pointers), then BFS from the start node
- The answer is the maximum BFS distance from start to any node

### Steps

1. **Build parent map**: DFS to record each node's parent
2. **BFS from start**: treat tree as undirected graph (left child, right child, parent are all neighbors)
3. **Track max distance**: BFS level count = infection time

## Solution

```python
from collections import deque

def amountOfTime(root, start):
    # 1. Build parent mapping
    parent = {}
    def dfs(node, par):
        if not node:
            return
        parent[node] = par
        dfs(node.left, node)
        dfs(node.right, node)

    dfs(root, None)

    # 2. Find the start node
    start_node = None
    def find(node):
        nonlocal start_node
        if not node:
            return
        if node.val == start:
            start_node = node
            return
        find(node.left)
        find(node.right)

    find(root)

    # 3. BFS to compute max distance
    visited = set([start_node])
    queue = deque([start_node])
    time = -1  # starts at -1 since start is already infected at minute 0

    while queue:
        time += 1
        for _ in range(len(queue)):
            node = queue.popleft()

            for neighbor in [node.left, node.right, parent.get(node)]:
                if neighbor and neighbor not in visited:
                    visited.add(neighbor)
                    queue.append(neighbor)

    return time
```

## Complexity

- **Time**: O(n) - DFS + BFS, each visits all nodes once
- **Space**: O(n) - parent map and BFS queue

## Key Points

1. **Graph conversion**: treat binary tree as undirected graph for infection spread
2. **BFS levels**: number of BFS levels equals infection time
3. **Parent tracking**: must record parent nodes to allow upward infection
4. **Start handling**: start node is infected at minute 0
