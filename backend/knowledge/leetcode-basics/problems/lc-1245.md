# LC 1245 - Tree Diameter

## Problem

Given an undirected tree with `n` nodes (labeled 0 to n-1) and an edge list, return the tree's diameter — the number of edges in the longest path between any two nodes.

## Approach

### Two BFS Passes

**Key theorem**: the tree diameter can be found with two BFS/DFS passes.

1. **First BFS**: start from any node, find the farthest node A
2. **Second BFS**: start from node A, find the farthest node B
3. Distance from A to B is the diameter

**Why this works**: from any starting node, the farthest node is always an endpoint of some diameter path (provable by contradiction).

## Solution

```python
from collections import defaultdict, deque
from typing import List

def treeDiameter(edges: List[List[int]]) -> int:
    if not edges:
        return 0

    graph = defaultdict(list)
    for u, v in edges:
        graph[u].append(v)
        graph[v].append(u)

    def bfs(start: int):
        """BFS from start, return (farthest_node, max_distance)"""
        visited = set([start])
        queue = deque([(start, 0)])

        farthest_node = start
        max_distance = 0

        while queue:
            node, dist = queue.popleft()

            if dist > max_distance:
                max_distance = dist
                farthest_node = node

            for neighbor in graph[node]:
                if neighbor not in visited:
                    visited.add(neighbor)
                    queue.append((neighbor, dist + 1))

        return farthest_node, max_distance

    # First BFS: find one endpoint of the diameter
    endpoint1, _ = bfs(0)

    # Second BFS: find the other endpoint and the diameter
    _, diameter = bfs(endpoint1)

    return diameter
```

```
BFS from 0: farthest = 5 (distance 3)
BFS from 5: farthest = 3 (distance 4)

Diameter: 4 (path: 3-2-1-4-5)
```

## Complexity

- **Time**: O(n) - two BFS passes, each O(V+E) = O(n)
- **Space**: O(n) - adjacency list, queue, visited set

## Key Points

1. **Tree property**: connected acyclic graph, n nodes have n-1 edges
2. **Diameter endpoint**: BFS from any node always finds a diameter endpoint
3. **Two passes required**: single BFS is not sufficient to find the diameter
4. **Edge cases**: empty tree (return 0), single edge (return 1)
