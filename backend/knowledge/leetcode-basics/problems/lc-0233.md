# LC 0233 - Number of Digit One

## Problem

Given an integer `n`, count the total number of digit `1` appearing in all non-negative integers less than or equal to `n`.

## Approach

### Core Idea

- Use digit DP or mathematical case analysis
- Count occurrences of digit 1 at each digit position, then sum
- For position k (0-indexed from the right), the count depends on three parts

### Mathematical Analysis

For a number like `abcde`, consider position k (right to left, 0-based):
Let `cur` = current digit, `high` = digits above, `low` = digits below, `base = 10^k`

1. **cur = 0**: `high * base`
2. **cur = 1**: `high * base + low + 1`
3. **cur > 1**: `(high + 1) * base`

### Digit DP Approach

Standard digit DP template with state: `(pos, count, tight, lead_zero)`, counting occurrences of 1 in each number.

## Complexity

- **Math approach**: O(log n) - iterate through each digit position
- **Digit DP approach**: O(log n * state count)
- **Space**: O(log n) - digit length

## Key Points

1. **Digit separation**: split the number by digit positions, process each independently
2. **Case analysis**: three cases based on the current digit value
3. **Formula derivation**: understand the mathematical meaning of each formula
4. **Generalization**: similar approach works for counting any digit 0-9
