# LC 0312 - Burst Balloons

## Problem

Given `n` balloons indexed 0 to n-1, each with a number on it in array `nums`. Bursting balloon `i` earns `nums[left] * nums[i] * nums[right]` coins, where `left` and `right` are adjacent balloons after previous bursts. Find the maximum coins collectible.

## Examples

```
Input: nums = [3,1,5,8]
Output: 167
Explanation:
nums = [3,1,5,8] -> [3,5,8] -> [3,8] -> [8] -> []
coins = 3*1*5 + 3*5*8 + 1*3*8 + 1*8*1 = 167
```

## Approach

### Interval DP + Reverse Thinking

**Key insight**: instead of thinking about which balloon to burst first, think about which balloon to burst **last** in each interval.

### State Definition

```
dp[i][j]: maximum coins obtainable in open interval (i, j)
```

Add virtual balloons with value 1 at both ends: `[1, ...nums..., 1]`

### State Transition

```
dp[i][j] = max(dp[i][k] + dp[k][j] + nums[i] * nums[k] * nums[j])
where k is the last balloon burst in interval (i, j)
```

When k is the last to burst, the left and right sub-intervals are already cleared, so i and j are adjacent to k, giving score `nums[i] * nums[k] * nums[j]`.

## Solution

```python
def maxCoins(nums: List[int]) -> int:
    n = len(nums)
    balloons = [1] + nums + [1]
    m = len(balloons)

    dp = [[0] * m for _ in range(m)]

    for length in range(2, m):
        for i in range(m - length):
            j = i + length
            for k in range(i + 1, j):
                dp[i][j] = max(dp[i][j],
                              dp[i][k] + dp[k][j] + balloons[i] * balloons[k] * balloons[j])

    return dp[0][m-1]
```

## Complexity

- **Time**: O(n^3) - three nested loops
- **Space**: O(n^2) - DP table

## Key Points

1. **Reverse thinking**: "last to burst" instead of "first to burst"
2. **Open interval**: dp[i][j] excludes boundaries i and j
3. **Virtual balloons**: add 1 at both ends to simplify boundary handling
4. **Traversal order**: iterate by interval length, then starting point
